## docker

- docker build -t [name] . => create docker image
- docker run [name] => run docker image
- docker run -p 8000:8000 [ chai]
- docker image ls => show running images

## aws

- ssh ubuntu@54.227.6.11 => access remote terminal from local
- pm2 ls => show me the server list
- pm2 start => start the server
- pm2 restart 0 => restart the server
- pm2 stop 0 => stop the server
